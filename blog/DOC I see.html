Terminology/术语

Namespace/命名空间

允许开发人员在一个独特，应用相关的名字的名称下捆绑所有功能的容器。
在javaScript中，命名空间只是一个包含方法，属性，对象的对象。

types/JS的类型有

  • Number
  • String
  • Boolean
  • Symbol (new in Edition 6)
  • Object
    • Function
    • Array
    • Date
    • RegExp
  • null
  • undefined

memory leak/内存泄漏

使用闭包的坏处是：在IE浏览器中它容易导致内存泄漏。
在IE中，每当在一个JavaScript对象和一个本地对象之间形成循环应用时，就会发生内存泄漏。

more：MDN:内存泄漏

Data property/数据属性

特征 数据类型 描述 默认值
[[Value]] 任何JavaScript类型 这个属性的数据值 undefined
[[Writable]] Boolean false:则该属性的[[Value]]特性不能被修改 false
[[Enumerable]] Boolean false:则不出现在for...in的枚举中 false
[[Configurable]] Boolean false:则该属性不能被删除，并且除[[Value]]和[[Writable]]以外的特征不能被修改 false

过时的属性(在ECMAScript 3定义的, 在ECMAScript 5被重命名)
属性 类型 描述
Read-only 任何JavaScript类型 ES5 [[Writable]] 属性的反状态(Reversed state)
DontEnum Boolean ES5 [[Enumerable]] 属性的反状态
DontDelete Boolean ES5 [[Configurable]] 属性的反状态

Accessor property/访问器属性

特征 数据类型 描述 默认值
[[Get]] 函数对象或者 undefined 该函数使用一个空的参数列表，能够在有权访问的情况下读取属性值。另见 get。 undefined
[[Set]] 函数对象或者 undefined 该函数有一个参数，用来写入属性值，另见 set。 false
[[Enumerable]] Boolean false:则不出现在for...in的枚举中 false
[[Configurable]] Boolean false:则该属性不能被删除，并且除[[Value]]和[[Writable]]以外的特征不能被修改 false

tips/注意

[src/href]属性

如果直接访问dom.[src/href]将获取到绝对的URL路径，如果想只想获取相对的URL，请使用dom.getAttribute(src/href,4)
